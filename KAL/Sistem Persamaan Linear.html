Sistem Persamaaan Linear

Persamaan linear adalah persamaan matematika yang menggambarkan hubungan linear antara variabel-variabelnya. Secara umum, persamaan linear dapat ditulis dalam bentuk:

\[ y=mx+b \]

Dalam konteks yang lebih umum, persamaan linear dapat memiliki lebih dari satu variabel:

\[ a_1x_1+a_2x_2+…+a_nx_n=b \]

Contoh Persamaan Linear

import numpy as np
A = np.array([[8,9,1,19],[1,2,3,9],[2,1,1,5]])
print(A)
[[ 8 9 1 19]
[ 1 2 3 9]
[ 2 1 1 5]]

Sistem persamaan tersebut adalah:

  • \( 8x_1 + 9x_2 + x_3 = 19 \)
  • \( x_1 + 2x_2 + 3x_3 = 9 \)
  • \( 2x_1 + x_2 + x_3 = 5 \)

Eliminasi Gauss

Metode ini mentransformasi sistem menjadi bentuk segitiga atas (upper triangular) untuk memudahkan substitusi balik.

Fungsi Operasi Baris Dasar (Python)

def RowSwap(A,k,l):
    B = np.copy(A).astype('float64')
    B[[k,l]] = B[[l,k]]
    return B

def RowScale(A,k,scale):
    B = np.copy(A).astype('float64')
    B[k] *= scale
    return B

def RowAdd(A,k,l,scale):
    B = np.copy(A).astype('float64')
    B[l] += B[k] * scale
    return B

Penerapan Langkah demi Langkah

  1. Swap Baris 0 & 1: Baris 1 menjadi pivot utama.
  2. Eliminasi Kolom 1: Membuat nol di bawah pivot baris pertama.
  3. Substitusi Balik: Setelah menjadi segitiga atas, hitung nilai variabel.
Hasil Akhir Matriks Tereliminasi:
[[ 1. 2. 3. 9. ]
[ 0. 1. 3.28 7.57 ]
[ 0. 0. 1. 2. ]]

Solusi Akhir:

\[ x_1 = 1, \quad x_2 = 1, \quad x_3 = 2 \]
